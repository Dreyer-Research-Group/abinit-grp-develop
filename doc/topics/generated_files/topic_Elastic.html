1. Introduction.

The DFPT theory to compute the elastic, piezoelectric and internal strain tensors is presented in [Hamann2005].

See the generalities about DFPT as well as the tutorial lesson_elastic. For the preliminary runs of ABINIT, see rfstrs, while for ANADDB, see elaflag for the elastic tensor, and piezoflag for the piezoelectric tensor.

    3. Related input variables.

    Compulsory input variables:

    ... rfatpol [Response Function : ATomic POLarisation]
    ... rfdir [Response Function : DIRections]
    ... rfstrs [Response Function with respect to STRainS]

    Basic input variables:

    ... elaflag@anaddb [ELAstic tensor FLAG]
    ... instrflag@anaddb [INternal STRain FLAG]
    ... piezoflag@anaddb [PIEZOelectric tensor FLAG]
